# 🧾 SlipSense

**SlipSense** คือแอปพลิเคชันจัดการการเงินบน iOS ที่ช่วยเปลี่ยน **"รูปสลิปโอนเงิน"** ในคลังภาพให้กลายเป็น **"บันทึกรายจ่ายอัตโนมัติ"** ได้ทันที โดยไม่ต้องพิมพ์เองแม้แต่ตัวอักษรเดียว แอปพลิเคชันถูกออกแบบมาโดยคำนึงถึงความเป็นส่วนตัวสูงสุด (Privacy-First) ข้อมูลทุกอย่างประมวลผลบนเครื่อง 100%

## ✨ ฟีเจอร์เด่น (Key Features)

- 📸 **On-Device OCR (สแกนสลิปออฟไลน์):** ดึงข้อมูลยอดเงิน วันเวลา และชื่อธนาคารจากรูปสลิป ด้วยอัลกอริทึมสแกน 2 รอบ (Two-Pass Scanning) เพื่อประมวลผลภาพและเพิ่มความแม่นยำในการอ่านภาษาไทย โดยข้อมูลไม่รั่วไหลออกนอกอุปกรณ์
- 🧠 **Self-Transfer Guard (ระบบกันยอดซ้ำ):** อัลกอริทึมดักจับและคัดกรองชื่อผู้โอน-ผู้รับ หากเป็นการโอนเงินระหว่างบัญชีของตัวเองแบบข้ามธนาคาร ระบบจะข้ามการคำนวณยอดนั้นอัตโนมัติเพื่อไม่ให้รายจ่ายซ้ำซ้อน
- 📊 **Smart Dashboard:** 
  - **Dynamic Donut Chart:** สรุปสัดส่วนการใช้จ่ายแยกตามธนาคาร
  - **Daily Trend:** กราฟแท่งสรุปเทรนด์การใช้เงินย้อนหลัง
  - **Time-Spend Analysis:** วิเคราะห์พฤติกรรมการใช้เงินตามช่วงเวลา (เช้า, บ่าย, ค่ำ, ดึก)
- 🔍 **Smart Search:** ค้นหาประวัติการโอนเงินจากข้อความบนสลิปได้อย่างรวดเร็ว (ชื่อ, ยอดเงิน, บันทึกช่วยจำ)
- 🌓 **Adaptive UI:** รองรับการปรับเปลี่ยนธีม Dark Mode / Light Mode แบบสมบูรณ์

## 🛠️ สถาปัตยกรรมและเทคโนโลยี (Architecture & Tech Stack)

โปรเจกต์นี้พัฒนาด้วย **Swift** แบบ Native 100% โดยยึดหลักการออกแบบ **MVVM (Model-View-ViewModel)** เพื่อแยก Business Logic ออกจาก UI อย่างเด็ดขาด ทำให้โค้ดสะอาด (Clean Code) บำรุงรักษาง่าย และพร้อมสำหรับการสเกล โครงสร้างหน้าจอทั้งหมดออกแบบโดยอิงตามมาตรฐาน **Apple Human Interface Guidelines (HIG)**

- **UI Framework:** `SwiftUI`
- **Local Database:** `SwiftData`
- **Data Visualization:** `Swift Charts`
- **Machine Learning / OCR:** `Vision` (VNRecognizeTextRequest)
- **Asset Management:** `PhotoKit`
- **System Integration:** `AppIntents` (Siri Shortcuts)
